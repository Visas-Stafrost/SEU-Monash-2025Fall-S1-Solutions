# 课程介绍

本课程编号为 01 ,因为按照考试时间顺序,这是 25 最早安排期中考试的课程，期中考试时间也比其他课早了一个多月。以下内容仅代表25级的真实体验。

## 授课教师

2025秋季学期任课教师为**方鹏飞**老师。

- **教学风格**: 课堂上比较严肃,但课后挺亲和,为人不错。
- **选导师建议**: 如果还没选择导师,方老师是值得考虑的人选(不过很抢手,需要抓紧)。

## 考勤情况

- 整个学期仅开学时点过一次名

## 课程内容

非常中规中矩的模式识别课程，但知识点不是很多，比起东南本科同样的课程少一半的内容。不过一些推导和计算对于理解现代模型还是很有帮助的。

## 考试情况

### 期中考试

- **题型**: 题目常年不变,使用仓库中的 `midterm-exam` 题
- **难度**: 难度较低，就三章内容也允许带一张 cheatpaper ，甚至会给出公式（但是期末相同题型是不给的）
- **成绩**: 据可靠消息,25期中考试全员满分

### 期末考试

- **题型**: 共六题，包含一条概念性简答题、四条计算题和一条偏开放实则大模型八股简答题
- **难度**: 总体难度不算难,由于可以带三张 cheatpaper 和课本（黑书）,没什么盲区存在，但是25期末那张试卷计算的数字没有设计过，计算量有点大，真考场上现学肯定来不及
- **建议**: 不要掉以轻心,但按照题型挨个过一遍混个眼熟，考场能在 cheatpaper 上找到就行
- **成绩**: 使用我们整理的资料复习的同学，这门课基本都 90 ，除了会长由于在考试时忘了翻页没看见最后一题痛失 15 分导致总分80。

## 复习指南

- **使用资料**: 个人最推荐的复习资料当属本部的homework，里面都是不同的题型。也就是仓库中的 `hw1` 等，对其中会考到的题型等进行了整理，删去了我们这里不讲的内容。 
- **复习建议**: 对照着题解把 homework 过一遍，对题型有疑问的，可以问AI。学有余力的同学可以过一遍ppt，或者看看知识点整理里的资料。

## cheatpaper

- **仓库介绍**: 仓库中 `cheatpaper` 里 L 文件夹下的是两次考试知识点包括部分例题的缩印全集，J 文件夹下的是期中考试后学习的知识点的题型整理步骤。实战其实可以都带，因为可以带书，可以写或者粘书上，老师也不太管，所以说甚至不需要缩印。
- **使用体验**: 以最后的考试论的话，理论上讲用 L 的 cheatpaper 上的知识可以保证你做出前五道题， J 的解题版有 3 、 4 两条一模一样的题，前两题是期中前的知识点，期中本协会学习的时候大伙都学过也没必要放。最后一题本来也有点老师夹带私货开放捞捞的感觉。
- **建议**: 属于本协会第一次合作准备考试，当时交流和排版还不是很成熟，会用 latex 但没想到可以用来搞这个。后续制作时可以参考算法课下的 cheatpaper 让AI使用latex排版，也可以排的很紧密，重点是有条理。虽然以回顾视角看， L 同学几次的 cheatpaper 都有行为艺术在里面的味道，但是感谢他的付出，到考前复习关头没什么人愿意花时间去做这个。
