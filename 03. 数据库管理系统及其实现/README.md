# 课程介绍

本课程编号为 03 ，本课程无考试，专业课里最水的一门。

## 授课教师

2025秋季学期任课教师为**崇志宏**老师。

- **老师介绍**: 吃尽时代红利的快退休的老大爷，研究方向比较偏数学，所以对计算机领域的观点都偏数学，对新技术了解不多。非常喜欢健身，身体素质在同年龄里可以排进前万分之一。
- **选导建议**: 对自己学生完全放养，毕业自己解决，但是放实习，如果有相信能靠自己毕业、想去实习的可以找他，他招生名额还挺多。

## 考勤情况

- 完全没有任何考勤。

## 课程内容

大概就是讲点离散数学看待数据的观点，对于数据库的介绍只停留在入门，其他则是每节课都在说现在研究生扩招导致学位变水了不是个好事云云。非常杂乱，上课属于脚踩西瓜皮滑到哪是哪。
