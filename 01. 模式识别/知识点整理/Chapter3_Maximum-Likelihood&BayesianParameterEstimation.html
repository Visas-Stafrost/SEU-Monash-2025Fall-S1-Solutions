Chapter 3: Maximum Likelihood & Bayesian Parameter Estimation

3.1 introduction

For Bayes Formula

P(wj∣x)=p(x∣wj)⋅P(wj)p(x)P(w_j|x) = \frac{p(x|w_j) \cdot P(w_j)}{p(x)} P(wj​∣x)=p(x)p(x∣wj​)⋅P(wj​)​

To compute posterior probability P(wj∣x)P(w_j|x)P(wj​∣x), we need to know Prior probability P(wj)P(w_j)P(wj​) and Likelihood p(x∣wj)p(x|w_j)p(x∣wj​).

For prior probability, there is no difficulty in computing: P(wj)=∣Dj∣∑i=1c∣Di∣P(w_j) = \frac{|\mathcal{D_j}|}{\sum_{i=1}^c} |\mathcal{D_i}|P(wj​)=∑i=1c​∣Dj​∣​∣Di​∣

However, class-conditional probability density function p(x∣wj)p(x|w_j)p(x∣wj​) is hard to assess. The main problems are listed below:

  • Training samples always seems to be numerically limited.
  • When the dimensionality of the feature space is high, here comes the problem of computaion complexity.

To solve the problem, we assume p(x∣wj)p(x|w_j)p(x∣wj​) has a certain parametric form.

  • Assumption 1: Maximum-Likelihood(ML) Estimation
    • View parameters as quantities whose value are fixed but unknown
    • Estimate parameter values by maximizing the likelihood (probability) of observing the actual training examples.
  • Assumption 2: Bayesian Estimation
    • View parameters as random variables having some known prior distribution.
    • Observation of the actual training examples transforms parameters' prior distribution into a posterior distribution (via Bayes theorem).

3.2 Maximum-Likelihood Estimation

3.2.1 Basic principle

Settings:

  • Likelihood function for each category is governed by some fixed but unknown parameters, i.e. p(x∣wj,θj)(1≤j≤c)p(x|w_j, \theta_j)(1 \leq j \leq c)p(x∣wj​,θj​)(1≤j≤c)

Task:

  • Estimate the parameters {θj}j=1c\{\theta_j\}_{j=1}^c{θj​}j=1c​ form the training data {Dj}j=1c\{\mathcal{D_j}\}_{j=1}^c{Dj​}j=1c​

Assuming that Examples in Dj\mathcal{D_j}Dj​ give no information about the parameters θj\theta_jθj​ if i≠ji \not ={j}i=j, each category can work separately hence we can simplify the notations Dj⇒D;θj⇒θ\mathcal{D_j} \Rightarrow \mathcal{D}; \theta_j \Rightarrow \thetaDj​⇒D;θj​⇒θ without loss of generality.

Given that:

xk∼p(x∣θ)x_k \sim p(x|\theta) xk​∼p(x∣θ)

  • θ\thetaθ: Parameters to be estimated
  • D\mathcal{D}D: A set of i.i.d. (independent and identically distributed) examples {x1,x2,...,xnx_1, x_2,..., x_nx1​,x2​,...,xn​}

Here we have the objective function, i.e. the likelihood of θ\thetaθ w.r.t. (with respect to) the set of observed examples:

p(D∣θ)=∏k=1np(xk∣θ)p(\mathcal{D}|\theta)= \prod_{k=1}^n p(x_k|\theta) p(D∣θ)=k=1∏n​p(xk​∣θ)

The maximum-likelohood estimation:

θ^=arg max⁡θp(D∣θ)\hat\theta = \argmax_{\theta} p(\mathcal{D}|\theta) θ^=θargmax​p(D∣θ)

  • Intuitively, θ^\hat\thetaθ^ best agrees with the actually observed examples.

Assuming the number of unknown parameters is ppp, let θ=(θ1,…,θp)∈Rd\theta=(\theta_1, \ldots,\theta_p) \in \mathbf{R}^dθ=(θ1​,…,θp​)∈Rd be a d-dimensional vector.

θ^=arg max⁡θp(D∣θ)⇔θ^=arg max⁡θl(θ)\hat\theta = \argmax_{\theta} p(\mathcal{D}|\theta) \Leftrightarrow \hat\theta=\argmax_{\theta} l(\theta) θ^=θargmax​p(D∣θ)⇔θ^=θargmax​l(θ)

3.3 Bayesian Estimation

Settings:

  • The parametric form of the likelihood function for each category is known p(x∣wj,θj)(1≤j≤c)p(x|w_j, \theta_j) \quad (1 \leq j \leq c)p(x∣wj​,θj​)(1≤j≤c)
  • θj\theta_jθj​ is considered to be random variables instead of being fixed (but unknown) values

In this case, we can no longer make a single ML estimate θ^j\hat\theta_jθ^j​. Instead, we fully exploit training samples to make the estimation.

P(wj∣x)⇒P(wj∣x,D∗)(D=D1∪D2∪⋯∪Dc)P(w_j|x) \Rightarrow P(w_j|x, \mathcal{D}^*) \qquad (\mathcal{D} = \mathcal{D_1} \cup \mathcal{D_2} \cup \cdots \cup \mathcal{D_c}) P(wj​∣x)⇒P(wj​∣x,D∗)(D=D1​∪D2​∪⋯∪Dc​)

3.3.1 class-conditional probability density function

Two assumptions:

  1. p(wj∣D∗)=p(wj)p(w_j|\mathcal{D}^*) = p(w_j)p(wj​∣D∗)=p(wj​)
  2. p(x∣wj,D∗)=p(x∣wj,Dj)p(x|w_j, \mathcal{D}^*) = p(x|w_j, \mathcal{D}_j)p(x∣wj​,D∗)=p(x∣wj​,Dj​): training samples in Di\mathcal{D_i}Di​ have no impact on calculating p(x∣wj,Dj)p(x|w_j, \mathcal{D_j})p(x∣wj​,Dj​) if i≠ji \not ={j}i=j

p(wj∣x,D∗)=p(wj,x,D∗)p(x,D∗)=p(wj,x,D∗)∑i=1cp(wi,x,D∗)=p(D∗)⋅p(wj∣D∗)⋅p(x∣wj,D∗)p(D∗)⋅∑i=1cp(wi∣D∗)⋅p(x∣wi,D∗)=p(wj∣D∗)⋅p(x∣wj,D∗)∑i=1cp(wi∣D∗)⋅p(x∣wi,D∗)=p(wj)⋅p(x∣wj,Dj)∑i=1cp(wi)⋅p(x∣wi,Di)\begin{aligned} p(w_j|x, \mathcal{D}^*) &= \frac{p(w_j,x,\mathcal{D}^*)}{p(x,\mathcal{D}^*)} = \frac{p(w_j,x,\mathcal{D}^*)}{\sum_{i=1}^c p(w_i,x,\mathcal{D}^*)} \\ &= \frac{p(\mathcal{D^*}) \cdot p(w_j|\mathcal{D^*}) \cdot p(x|w_j, \mathcal{D^*})}{p(\mathcal{D^*}) \cdot \sum_{i=1}^c p(w_i|\mathcal{D^*}) \cdot p(x|w_i, \mathcal{D^*})} \\ &= \frac{p(w_j|\mathcal{D^*}) \cdot p(x|w_j, \mathcal{D^*})}{\sum_{i=1}^c p(w_i|\mathcal{D^*}) \cdot p(x|w_i, \mathcal{D^*})} \\ &= \frac{p(w_j) \cdot p(x|w_j, \mathcal{D_j})}{\sum_{i=1}^c p(w_i) \cdot p(x|w_i, \mathcal{D_i})} \end{aligned} p(wj​∣x,D∗)​=p(x,D∗)p(wj​,x,D∗)​=∑i=1c​p(wi​,x,D∗)p(wj​,x,D∗)​=p(D∗)⋅∑i=1c​p(wi​∣D∗)⋅p(x∣wi​,D∗)p(D∗)⋅p(wj​∣D∗)⋅p(x∣wj​,D∗)​=∑i=1c​p(wi​∣D∗)⋅p(x∣wi​,D∗)p(wj​∣D∗)⋅p(x∣wj​,D∗)​=∑i=1c​p(wi​)⋅p(x∣wi​,Di​)p(wj​)⋅p(x∣wj​,Dj​)​​
